Jupyter
=======

> Установка и настройка для работы с python2, python3, NodeJS.

##### Установить
    pip install jupyter

##### Добавить Python2
    python2 -m pip install ipykernel

##### Добавить Python3
    python3 -m pip install ipykernel

##### Добавить NodeJS
    npm install -g ijavascript 
    ijsinstall

##### Запустить
    jupyter notebook
